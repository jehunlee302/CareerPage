# UI 요구사항

## 화면 목록

### 인증
- 로그인: 이메일+비밀번호, 소셜 버튼 (Google, Kakao)
- 회원가입: 이메일, 비밀번호, 비밀번호 확인
- 비밀번호 재설정: 이메일 입력 → 확인 메시지

### 상품
- 목록: 그리드 뷰, 필터 (카테고리, 가격), 정렬, 무한 스크롤
- 상세: 이미지 갤러리, 설명, 가격, 구매 버튼
- 등록/수정: 폼 (이미지 업로드 드래그앤드롭)

### 주문
- 주문 생성: 배송지 입력, 결제 수단 선택
- 주문 목록: 상태별 탭 필터
- 주문 상세: 상태 타임라인, 취소 버튼

### 대시보드
- 판매자: 매출 차트, 주문 현황, 상품 관리
- 구매자: 주문 이력, 위시리스트

## 상태 처리 (모든 화면 필수)
- Loading: 스켈레톤 UI
- Error: 에러 메시지 + 재시도 버튼
- Empty: 빈 상태 일러스트 + 안내 메시지

## 반응형
- Breakpoints: 640px (모바일), 1024px (태블릿), 1280px (데스크톱)
