## Design Decisions

| Decision | Rationale |
|----------|-----------|
| Vanilla JS (no framework) | Zero dependency, GitHub Pages compatible, fast load |
| `data/portfolio.json` as single source | Decouples content from presentation |
| `onEdit` Apps Script trigger | Instant sync without manual steps |
| Dark hero + dark philosophy section | Brand continuity; philosophy is prominent as requested |
| Space Grotesk for headings | Modern, technical, distinctive — fits AI/engineering profile |
| Philosophy section immediately after hero | User's "Systemic Excellence" philosophy is prominent to all visitors |

## Pending Setup (by user)

1. Push repo to GitHub (`your-username/your-repo-name`)
2. Enable GitHub Pages (Settings > Pages > Source: GitHub Actions)
3. Configure GoDaddy DNS (A records → GitHub Pages IPs, CNAME www → `username.github.io`)
4. Open Google Sheet > Extensions > Apps Script > paste `scripts/google-apps-script.js`
5. Set Script Properties: `GITHUB_TOKEN`, `GITHUB_REPO`
6. Run `setupTrigger()` once
